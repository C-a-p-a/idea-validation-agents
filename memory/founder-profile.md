- **Small home market** (~5.5M; Nordics ~27M). Model Norway/Nordics SOM first, then EU / English-global for scale.
- **Funding:** grant-driven culture (Innovasjon Norge, SkatteFUNN, Forskningsrådet) + bootstrapping over US-style VC.
